Description

interActive Window explores the shop window as an architectural threshold — the boundary between street and interior. The installation treats this boundary as a set of interdependent parameters, using sensing technology, urban flow, and kinetic effects to produce a space that responds to passers-by and connects heterogeneous everyday objects into a single dynamic system.

Marilyn Reloaded

The work shown here, Marilyn Reloaded, was presented within the interActive Window exhibition — a mannequin draped in a dark, bag-like garment suspended above a desk fan and wall text, set inside a street-level gallery vitrine in Vienna.
